# Deep Workflow

类 dify、coze 的可视化流程图引擎，支持变量、判断、代码块、循环等，适用于大模型、AI Agent 等场景，支持自由扩展节点。

<iframe src="/dist/index.html" style="width: 100%; height: 600px; border: none;"></iframe>

## 特性
- 可视化工作流编辑
- 节点变量与运行日志
- 错误聚焦与中断恢复
- 节点单测与失败重试
- 历史记录、撤销重做
- 节点插件机制（规划中）
- 完善的 API 与扩展能力

## 核心理念
- 简单易用，开箱即用
- 灵活扩展，适配多场景
- 关注开发者体验
